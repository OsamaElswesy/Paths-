name: backend-audit
description: Audit the current FastAPI backend implementation.
---

When auditing backend:
- identify main modules and responsibilities
- inspect APIs, services, models, orchestration, prompts, workflows, integrations, and logging
- determine what is complete, partial, missing, or risky
- check separation of concerns
- check error handling and testability
- check whether agent logic is explicit or scattered
- return exact file paths when relevant
