---
name: agent-discovery
description: Discover what agents already exist in the project codebase.
---

When discovering agents:
- search for agent-related modules, prompts, orchestration logic, workflows, tools, handlers, and services
- identify each agent's current purpose
- identify whether each agent is implemented, partial, or missing
- identify overlaps and unclear boundaries
- do not invent missing implementations
- prefer code truth over project document assumptions
