name: frontend-audit
description: Audit the current Next.js frontend implementation.
---

When auditing frontend:
- identify routes, pages, components, state management, API integration, and workflow UI
- determine what is complete, partial, missing, or risky
- inspect human-in-the-loop screens and recruiter workflows
- check loading, error, and empty states
- check maintainability and component structure
- return exact file paths when relevant
